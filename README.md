# Infrastructure Monitoring Dashboard using Grafana, Prometheus & Node Exporter

## Overview

This project demonstrates the implementation of a complete monitoring and observability solution for Linux virtual machines using Prometheus, Grafana, and Node Exporter.

The monitoring stack collects real-time infrastructure metrics and visualizes them through interactive Grafana dashboards, enabling system administrators and DevOps engineers to monitor server health, identify bottlenecks, and troubleshoot performance issues efficiently.

## Features

- Real-time CPU monitoring
- Memory utilization tracking
- Disk usage monitoring
- Network traffic analysis
- Interactive Grafana dashboards
- Prometheus-based metrics collection
- Node Exporter integration
- Infrastructure observability
